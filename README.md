## üìå Overview
This is an interactive **Business Intelligence (BI) Dashboard** built to analyze sales performance, profit margins, and customer trends for a fictional retail superstore. 

The application transforms raw sales data into actionable insights using dynamic visualizations, enabling users to track Key Performance Indicators (KPIs) in real-time.

## üöÄ Key Features
* **KPI Tracking:** Real-time monitoring of Total Sales ($), Total Profit ($), and Profit Margins (%).
* **Interactive Filters:** Sidebar controls to slice data by **Region** (North, South, East, West) and **Product Category**.
* **Dark Mode UI:** Custom CSS styling for a modern, professional "Dark Mode" aesthetic with high-contrast metric cards.
* **Dynamic Visualizations:**
    * **Bar Chart:** Top-performing product sub-categories.
    * **Donut Chart:** Regional sales distribution.
    * **Area Chart:** Monthly revenue trends over time.
